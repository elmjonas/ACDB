----------------------------------------------------------------------------------------------------------
"benchmark_data.csv" contains the energies for the benchmark used in Figures 1-4 and  SI 1-2.
>> All values are in Hartree
>> basename refers to the structurefile name in benchmark_structures.zip
>> F12_VTZ_FULL is the CCSD(F12*)(T+)/cc-pVTZ-F12 reference values
>> DLPNO is DLPNO-CCSD(T) and LNO is LNO-CCSD(T)
>> "SCF" is the SCF energy, "CORR" is the correlation energy, and "FULL" is the total electronic energy. 
----------------------------------------------------------------------------------------------------------
"benchmark_logs.zip" contains the log files for the LNO-CCSD(T), DLPNO-CCSD(T), CCSD(T), and CCSD(F12*)(T+) calculations for the dimer set. 
----------------------------------------------------------------------------------------------------------
"benchmark_structures.zip" contains the xyz files for the dimer set. 
----------------------------------------------------------------------------------------------------------
"actual_free_energies.csv" contains the free energies under given conditions for the (SA)1-4(AM/DMA)1-4 clusters.
 >> All values are in kcal/mol
 >> All values are binding free energies with the single-point at either the DLPNO-CCSD(T)/aug-cc-pVTZ or LNO-CCSD(T)/CBS(,) level and the thermal part at the wB97X-D/6-31++G(d,p) level. 
 >> DLPNO is the DLPNO/CCSD(T) method, the CBS columns are the LNO calculations.  
----------------------------------------------------------------------------------------------------------
 "15sa15tma_data.zip" contains the log files and structures for the (SA)15(TMA)15 calculations and its monomers. 
----------------------------------------------------------------------------------------------------------
 "acdc_structure_data.zip" contains the log files and structures for the (SA)1-4(AM/DMA)1-4 clusters used for the ACDC calculations
