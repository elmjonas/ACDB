Supporting information for the following article: 
Humidity effect on the first steps of atmospheric new particles formation: Computational study of hydrated molecular clusters

Each of the folders supports pickle files (see the JK framework or ACDB manual) for given article section:

- el_energy_benchmark = electronic energy benchmark
- el_energy_error_scaling = electronic energy error scaling with system size
- opt_geometries = optimized geometries
- vibrational_frequencies = vibrational frequencies of optimized structures
- hydration_distribution = hydration distribution 
- binding_free_energies = binding free energies

For more information please visit: https://github.com/elmjonas/ACDB
